筷子 （100 分）

hwj 有一些筷子。他将筷子摆在 X 轴上的某些位置，有些筷子有重叠。现在他想去掉一些筷子，使筷子不重叠，请问最少要去掉多少根筷子？

输入格式

输入文件名为 chopsticks.in。
第一行一个数 n，表示筷子数量。之后 n 行，每行两个数，表示每根筷子的开始坐标和结束坐标。

输出格式

输出文件名为 chopsticks.out。
输出一行，表示最多能放几根筷子。

样例输入

5
2 4
1 12
4 5
7 10
7 8

样例输出

2

数据范围

  • 对于 30% 的数据，1 ≤ n ≤ 20。
  • 对于另外 20 % 的数据，1 ≤ si, ei ≤ 1000。
  • 对于 100% 的数据，1 ≤ n ≤ 100000，1 ≤ si, ei ≤ 100000000。

超级数（100 分）

神 lyh 闲来无事，创造了一种概念叫做超级数对。当 a 能通过某种特定的变换方式变成 b，则称 a 和 b 为一对超级数对。

为了完成变换，你可以执行以下两种操作：
* 将当前的数的二进制最右边加一个 0。（将x变为 2x）
* 在当前的数的十进制最右边加一个 1。（将x变为 10x+1）

lyh 发明了超级数对之后突然想研究{理，他给出了两个数 a 和 b，他想知道将 a 通过这两种操作变为 b 并输出最小的操作次数与中间过程的操作序列，或者说 a 和 b 并不是超级数对。

输入

输入文件名为 super.in。
输入数据共一行两个正整数 a, b。

输出

输出文件名为 super.out。
如果 a 和 b 不是超级数对，输出 NO，否则输出三行：
第一行输出 YES。
第二行有一个整数 k 最小的操作序列长度。
第三行输出操作序列 x1, x2, x3 .. xk。x1 = a, xk = b。x_i 由 x_i-1 通过上述两种操作变化而来。

输入样例 1

2 162

输出样例 1

YES
5
2 4 8 81 162

输入样例 2

4 42

输出样例 2

NO

数据范围

对于 30% 的数据 1 ≤ a < b ≤ 10^3；
对于 80% 的数据 1 ≤ a < b ≤ 10^9；
对于 100% 的数据 1 ≤ a < b ≤ 10^18。
